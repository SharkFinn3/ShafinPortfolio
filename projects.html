NaturalNumber Calculator GUI

The NaturalNumber Calculator GUI is a Java-based calculator application developed using the Swing framework and follows the model-view-controller (MVC) design pattern to perform arithmetic operations specifically designed for natural numbers.

The model component encapsulates the business logic for performing arithmetic calculations, while the view component represents the GUI elements, enabling user interaction. The controller component handles user inputs and orchestrates the communication between the model and view.

One of the notable aspects of this project is the incorporation of robust error handling and input validation mechanisms. This ensures that users receive informative and context-specific messages for invalid input, enhancing the overall usability and user experience of the calculator.

Technologies used: Java, Swing.

Aviation Weather App

Web-based application that provides real-time weather information for aviation enthusiasts and professionals.

The app uses the Requests library to make HTTP requests to the Aviation Weather API, fetching the latest METAR (Meteorological Aerodrome Report) data for the specified station.

Utilizing the XML module to parse the received XML data and extract the relevant weather information.

Technologies used: Python, Tkinter, XML parsing, API Requests.
